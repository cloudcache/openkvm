openkvm
=======

简单轻量级的开源KVM虚拟化管理平台

需求：
1.物理机管理：  
  1.1 增加物理机  
  1.2 查看物理机  
  1.3 编辑物理机  
  1.4 删除物理机  
  1.5 列出物理机列表  

2.存储管理  
  2.1 增加存储  
  2.2 查看存储  
  2.3 编辑存储  
  2.4 删除存储  
  2.5 存储列表  
  2.6 备份任务管理  
      2.6.1 增加备份任务  
      2.6.2 查看备份任务  
      2.6.3 编辑备份任务  
      2.6.4 删除备份任务  
      
3.虚拟网络管理  

4.监控  

5.虚拟机管理  

6.API接口
